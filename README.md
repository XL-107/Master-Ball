# Master Ball
Master Ball is a flutter-based Android app all about Pokemon! 
We plan for this pokemon database to have sorting, display, and also teambuilding capabilities. 

## Current Capabilites:
We have a basic understanding of flutter, but have not yet assembled a final product. 

### NOTES:
Try not to mix linux and windows installations when working on the project. 
The android emulator doesn't like being run out of WSL very muchc.
